COUNTRY NAMES
Each country name is listed using the format: 
[short country name] < [full country name] 
If the short name is used without **"the"** in a list of country names but with "the" in a sentence, it is followed by "[the]".
Some country names take different forms depending on whether they re in a list of country names (such lists should always be alphabetically sorted), or in a sentence. They appear below in the form used in lists. The sentence form moves the bracketed term to the front of the short name (without brackets).
**Example:**
Bolivia (Plurinational State of) < the Plurinational State of Bolivia
In this example, "Bolivia (Plurinational State of)" should appear in that form in lists of countries (sorted under the letter "b"). Its sentence form is "Plurinational State of Bolivia".
**Enforce the use of the short country name and correct any spelling and capitalization errors in the names. In other words, change any formal names to the short names. For example, if the text contains "Republic of Azerbaijan", correct it to "Azerbaijan".
If a country name is misspelled, correct it to the short country name listed below.
